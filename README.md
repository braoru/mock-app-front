Mock App Front-end
==================

## Install Angular CLI
```
npm install -g @angular/cli
```

## Generate code from scratch
### Initialize Angular project and install dependencies
```
ng new mock-app-front
cd mock-app-front
npm install -g @ng-bootstrap/ng-bootstrap
```
### Add a component
```
cd mock-app-front
ng generate component <...>
```

## Build
```
cd mock-app-front
ng serve --open
```
